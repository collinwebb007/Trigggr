How it Works

The technical journey of a physical signal mapped to a digital video frame.

1

Input: Universal Signal Capture

The Node acts as an intelligent intermediary, physically intercepting digital signals at the hardware layer to ensure absolute compatibility regardless of the enterprise stack.

  • USB HID (Human Interface Device) Packets
  • Serial RS-232/RS-485 Monitoring
  • Dry Contact Pulses
2

Processing: Sub-15ms Signal Formatting

To achieve frame-perfect synchronous video tagging, the node utilizes a dedicated RTOS thread to process incoming packets, encode the payload into XML, and broadcast it to the network in under 15 milliseconds.

Latency Profile: < 15.0ms (Deterministic)
3

Output: ONVIF Metadata Overlay

The normalized XML payload is injected directly into existing IP camera streams as searchable bookmarks.

Profile S Video streaming & PTZ metadata sync.
Profile G Edge storage recording index triggers.
Profile T Advanced event analytics binding.
